# Documentación de la API

Este espacio está reservado para la salida generada por herramientas como `flasgger`, `apispec` o `spectree`, y también para guías escritas manualmente sobre los endpoints expuestos por la API Flask.

Sugerencias:
- Publicar aquí el `openapi.yaml` exportado mediante `scripts/export_openapi.py`.
- Mantener ejemplos de requests/responses reales para los dominios principales (despachos, facturas, zonas francas).
- Documentar reglas de negocio relevantes y códigos de error.
