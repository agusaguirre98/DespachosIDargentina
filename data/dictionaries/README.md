# Dictionaries

Aquí se guardan diccionarios de referencia (catálogos, tablas maestras y mapas de códigos) necesarios para enriquecer los datos de despachos.

Usos habituales:
- Relacionar códigos SAP/aduana con descripciones legibles.
- Normalizar unidades de medida o tipos de gasto.
- Mantener listas de control para validaciones previas al ingreso de datos.

Procura versionar archivos en formato abierto (CSV, JSON, YAML) para facilitar su trazabilidad.
