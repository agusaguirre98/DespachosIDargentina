# Samples

Este directorio almacena archivos de ejemplo utilizados para pruebas manuales y demostraciones de la plataforma.

- **PDFs de despacho**: permiten ejercitar los extractores OCR y los pipelines de normalización.
- **Archivos CSV/Excel**: útiles para simular importaciones o sincronizaciones masivas.

Ningún dato sensible debe subirse aquí; utiliza información ofuscada o generada.
